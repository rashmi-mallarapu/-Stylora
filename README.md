Stylora 👗✨
Fashion Assistance Web Application

📌 Problem Statement
Users struggle with outfit selection due to poor wardrobe organization, limited visualization, and lack of personalized fashion guidance.

📌 Proposed Solution
Stylora provides a fashion assistance platform that manages wardrobes digitally and offers occasion‑based outfit suggestions using predefined logic.

⚙️ How It Works
Users upload clothes, select occasions, and receive outfit suggestions generated through predefined rules and stored images.

🧩 Key Features
Virtual wardrobe, try‑on simulation, occasion planner, outfit comparison, color palette analysis, and sketch‑to‑style.

🧪 Demo Explanation
The demo shows outfit uploads, predefined suggestions, virtual previews, and comparisons.

🌐 Deployment
Deployed using GitHub Pages.

🛠️ Tech Stack
HTML, CSS, JavaScript, GitHub Pages.

✅ Conclusion
Stylora demonstrates rule‑based fashion assistance using predefined logic and visuals.
